  • Birds Species Detection using VGG 16 Algorithm

    • The Project proposes a robust deep learning BCC.h5 model by using VGG(Visual Geometry Group)-16 which is a deep convolution neural network for easily identifying bird species. Moreover, the incorporation of bird vocalizations introduces an immersive element to the project, enabling users to interact with the auditory facets of bird recognition
